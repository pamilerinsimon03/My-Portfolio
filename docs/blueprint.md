# **App Name**: My Portfolio

## Core Features:

- Hero Section: Display an introduction section with a brief personal summary and key skills.
- Project Showcase: Visually showcase projects with image/video previews, descriptions, tech stacks used, and links to live demos or GitHub repos.
- Experience Timeline: Interactive display of professional experience including company name, job title, dates, and key responsibilities.
- Skills Overview: Dedicated space to list technical skills and tools (languages, frameworks, libraries) with proficiency levels.
- Contact Section: Display contact information and a contact form.
- Tech Stack: Built with React.js and Tailwind CSS

## Style Guidelines:

- Primary color: Deep bright blue (#4169E1), echoing your requested color scheme and offering a visually striking, professional feel.
- Background color: Light blue-gray (#D8DDE6) to provide a muted backdrop that accentuates content and allows the primary color to stand out. Chosen for a light color scheme.
- Accent color: Electric Indigo (#6F00FF) for highlights, calls to action, and interactive elements. Its strong contrast against the light background will attract attention.
- Body font: 'Inter', a sans-serif font for its clean and modern appearance; pairs well with 'Space Grotesk' for the headline.
- Headline font: 'Space Grotesk', a sans-serif font for its clean and modern appearance; pairs well with 'Inter' for the body.
- Use a set of modern and minimalist icons for skills and tech stacks.
- Implement subtle and fluid animations for transitions and interactions. Focus on scroll-based animations and micro-interactions.
